# IT-Expenditure-Analysis

## Project Overview

This project analyzes organizational IT spending using Power BI. The dashboard provides insights into expenditure trends, budget utilization, vendor spending, and departmental cost allocation.

## Objectives

* Monitor overall IT expenditure.
* Compare actual spending against budget.
* Identify departments with the highest costs.
* Analyze vendor-wise spending patterns.
* Discover opportunities for cost optimization.

## Tools Used

* Power BI Desktop
* Microsoft Excel
* DAX

## Dashboard Pages

1. Executive Overview
2. Department Analysis
3. Vendor Analysis
4. Infrastructure & Software Costs
5. Insights & Recommendations

## Key Insights

* Total IT expenditure and budget variance.
* Department-wise spending distribution.
* Vendor contribution analysis.
* Infrastructure versus software cost comparison.
* Cost optimization recommendations.
